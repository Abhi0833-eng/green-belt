# Reward Token Contract

Custom Soroban token that rewards voters on the Advanced Poll dApp.

## Contract Address (Testnet)
`CAXZHPXTDHYDIGKOMNDP4FCJQF7HWSOEIC7SRWCSOIAP3EGVSMBXBQVO`

## Functions
- `initialize(admin)` - Setup contract
- `mint(to, amount)` - Mint tokens to address  
- `balance(addr)` - Get token balance
- `transfer(from, to, amount)` - Transfer tokens
- `total_supply()` - Get total supply
